# METAR/TAF Вкладка — Руководство пользователя

## 📋 Описание

Вкладка **METAR/TAF** в дашборде MIRA предоставляет фактические метеорологические данные с ближайших аэропортов в радиусе до 500 км от выбранной точки.

## 🎯 Возможности

### 1. Автоматическая загрузка данных
- Поиск ближайших аэропортов по координатам маршрута или центра карты
- Получение METAR (фактическая погода) и TAF (прогноз) для каждого аэропорта
- Кэширование данных на 60 минут для METAR и 6 часов для TAF

### 2. Оценка актуальности
- **🟢 Актуально** — данные моложе 30 минут
- **🟡 Устарело** — данные от 30 до 60 минут
- **⚠️ Старые данные** — старше 60 минут (не рекомендуются для использования)

### 3. Отображаемые параметры

| Параметр | Описание |
|----------|----------|
| **Аэропорт** | ICAO код, название, город |
| **Расстояние** | До точки поиска (км) + направление (румб) |
| **Время (UTC)** | Время наблюдения в формате ДД.ММ ЧЧ:ММ UTC |
| **Ветер** | Направление (°) + скорость (м/с) + порывы |
| **Видимость** | В метрах или километрах |
| **Погода** | Явления (дождь, снег, туман и т.д.) |
| **Облачность** | Количество ярусов + высота нижней границы |
| **Температура** | Температура воздуха + точка росы |
| **Давление** | QNH в гПа + мм рт.ст. |
| **Статус** | Актуальность данных |

### 4. Визуализация

Для каждого аэропорта отображаются:
- **Таблица** со всеми параметрами
- **Карточка** с подробными данными
- **Мини-роза ветров** (направление стрелкой)
- **METAR/TAF сырые данные** (в формате ICAO)

## 🚀 Как использовать

### Вариант 1: Автоматическая загрузка
1. Откройте дашборд (кнопка "ДАШБОРД" в правом верхнем углу)
2. Выберите вкладку **METAR/TAF** (вторая слева)
3. Если данные не загружены — нажмите кнопку **"Загрузить METAR для ближайших аэропортов"**
4. Система автоматически определит координаты из маршрута или центра карты

### Вариант 2: После анализа маршрута
1. Выполните анализ маршрута через основной интерфейс
2. Откройте дашборд
3. Перейдите на вкладку METAR/TAF
4. Данные загрузятся автоматически для начала маршрута

## 📊 Статистика

В верхней части вкладки отображается статистика:
- **Всего аэропортов** — найдено в радиусе поиска
- **Актуальные METAR** — данные моложе 60 минут
- **Устаревшие METAR** — данные старше 60 минут
- **Есть TAF** — аэропорты с прогнозом

## 🔧 Действия

### Обновить
Повторный запрос данных для всех аэропортов

### Экспорт JSON
Скачивает файл с данными в формате JSON:
```json
{
  "timestamp": "2026-03-05T12:00:00Z",
  "searchCenter": {"lat": 55.75, "lon": 37.62},
  "searchRadiusKm": 500,
  "airports": [...]
}
```

### Печать
Формирует печатную версию отчёта со всеми аэропортами

## 🎨 Цветовая схема

| Цвет | Значение |
|------|----------|
| 🟢 Зелёный | Актуальные данные (< 30 мин) |
| 🟡 Жёлтый | Устаревшие данные (30-60 мин) |
| 🔴 Красный | Критические условия (LIFR) |
| 🟠 Оранжевый | Сложные условия (IFR) |

## 📝 Пример METAR

```
UUEE 051200Z 260050 9999 FEW020 SCT100 12/08 Q1013 NOSIG
```

Расшифровка:
- **UUEE** — аэропорт Шереметьево
- **051200Z** — 5 число, 12:00 UTC
- **260050** — ветер 260° (западный), 5 м/с
- **9999** — видимость ≥10 км
- **FEW020** — мало облаков на высоте 2000 футов
- **SCT100** — разорванные облака на высоте 10000 футов
- **12/08** — температура +12°C, точка росы +8°C
- **Q1013** — давление 1013 гПа
- **NOSIG** — без значимых изменений

## ⚠️ Ограничения

1. **Частота обновления** — METAR обновляется каждые 30-60 минут
2. **Доступность** — не все аэропорты имеют METAR/TAF
3. **Задержка** — данные могут запаздывать на 10-20 минут
4. **CORS** — некоторые источники могут блокировать запросы из браузера

## 🔗 Источники данных

1. **METARTAF.RU** — основной источник для РФ
2. **NOAA Aviation Weather** — резервный источник (мир)
3. **CheckWX API** — дополнительный источник (при наличии ключа)

## 💡 Советы

- Используйте данные METAR для предполётного планирования
- Сравнивайте METAR с модельными данными (вкладка "Метеопрогноз")
- При отсутствии данных в радиусе 500 км — уменьшите радиус или выберите другую точку
- Для полётов в удалённых районах используйте данные ближайшего доступного аэропорта с поправкой на расстояние

## 🆘 Устранение неполадок

| Проблема | Решение |
|----------|---------|
| Нет данных | Проверьте интернет-соединение, попробуйте меньший радиус |
| Все данные устарели | Нажмите "Обновить", подождите 1-2 минуты |
| Ошибка CORS | Используйте браузер с отключенной CORS-защитой или серверный прокси |
| Медленная загрузка | Уменьшите количество аэропортов (максимум 12) |
