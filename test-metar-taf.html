MIRA — METAR/TAF Тест

Получение и анализ авиационных метеорологических данных из METARTAF.RU (основной) и NOAA Aviation Weather API (резервный)

О METAR и TAF
  • METAR — фактическая погода на аэродроме (обновление каждые 30-60 мин)
  • TAF — аэродромный прогноз на 24-30 часов (обновление каждые 6 часов)
  • VFR/MVFR/IFR/LIFR — категории лётных условий
  • Основной источник: ✈️ METARTAF.RU (для аэропортов РФ)
  • Резервный источник: 🌐 NOAA Aviation Weather API (через CORS-прокси)
⚠️ Внимание: METARTAF.RU может не работать из браузера из-за CORS.
  • Откройте страницу через локальный веб-сервер: node server.js
  • Или используйте start-metar-test.bat
  • При проблемах данные автоматически загрузятся из NOAA (через прокси)
