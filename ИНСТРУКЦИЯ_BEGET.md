# 📦 Развёртывание MIRA на Beget

## 🎯 Что нужно сделать

### 1. Регистрация на Beget

1. Перейдите на https://beget.com
2. Зарегистрируйтесь (бесплатный тариф)
3. Дождитесь активации аккаунта

---

### 2. Структура файлов на Beget

```
/ваш-домен.ru/
├── index.html              ← Главный файл
├── css/
│   ├── common.css
│   ├── desktop.css
│   └── dashboard-modal.css
├── js/
│   ├── app.js
│   ├── weather.js
│   ├── metar-taf.js
│   ├── vertical-profile.js
│   ├── verification.js
│   ├── atmospheric-indices.js
│   ├── interpolator.js
│   └── ... (остальные JS файлы)
├── api/
│   ├── metar.php          ← Прокси для NOAA ✅ НОВЫЙ
│   └── metartaf.php       ← Прокси для metartaf.ru ✅ НОВЫЙ
├── airports.json
└── ... (остальные файлы)
```

---

### 3. Загрузка файлов

#### Способ 1: Через файловый менеджер Beget

1. Войдите в панель управления Beget
2. Перейдите в **Файловый менеджер**
3. Откройте папку вашего домена (например, `public_html`)
4. Загрузите все файлы проекта

#### Способ 2: Через FTP

1. Скачайте FTP-клиент (FileZilla, WinSCP)
2. Подключитесь к серверу:
   - **Сервер:** `ftp.beget.com`
   - **Логин:** ваш логин Beget
   - **Пароль:** ваш пароль Beget
3. Загрузите файлы в папку `/public_html/`

---

### 4. Проверка работы API

После загрузки проверьте работу прокси:

```bash
# Проверка METAR прокси
https://ваш-домен.ru/api/metar.php?icao=UUDD

# Проверка METARTAF прокси
https://ваш-домен.ru/api/metartaf.php?icao=UUDD
```

**Ожидаемый ответ:**
```json
{
  "success": true,
  "icao": "UUDD",
  "timestamp": "2026-03-05T17:30:00+03:00",
  "data": [...],
  "count": 1
}
```

---

### 5. Настройка CORS (если нужно)

Если сайт будет доступен с другого домена, добавьте в начало `metar.php`:

```php
// Разрешить только вашему домену
$allowedOrigin = 'https://ваш-домен.ru';
header("Access-Control-Allow-Origin: $allowedOrigin");
```

Или для всех (уже настроено):
```php
header('Access-Control-Allow-Origin: *');
```

---

### 6. Проверка на сайте

1. Откройте `https://ваш-домен.ru`
2. Создайте маршрут
3. Откройте дашборд → вкладка **METAR/TAF**
4. Данные должны загрузиться автоматически

## 🔧 Требования Beget

| Параметр | Значение |
|----------|----------|
| **PHP** | 7.4 или выше (по умолчанию 8.x) |
| **cURL** | Включён (по умолчанию) |
| **SSL** | Бесплатный Let's Encrypt |
| **База данных** | Не требуется |

## 🐛 Возможные проблемы

### 1. Ошибка 500 на API

**Решение:** Проверьте версию PHP (нужна 7.4+)

```bash
# В панели Beget → PHP
Выберите PHP 8.0 или выше
```

### 2. cURL не работает

**Решение:** Включите в настройках PHP

```ini
; В .htaccess или php.ini
extension=curl
```

### 3. CORS ошибка

**Решение:** Проверьте заголовки в `metar.php`

```php
header('Access-Control-Allow-Origin: *');
```

### 4. Файлы не загружаются

**Решение:** Проверьте квоту (на бесплатном тарифе 2GB)
