# Промпт: Реализация печати отчётов в веб-приложении

**Задача:** Реализовать функционал печати отчётов и экспорта в PDF для веб-приложения.

---

## Требования

### 1. Кнопка печати

Добавить кнопку «🖨️ Печать / PDF» с вызовом `window.print()`:

```html
<button class="btn-secondary" onclick="window.print()">🖨️ Печать / PDF</button>
```

Разместить в видимой области интерфейса.

---

### 2. Стили для печати (`@media print`)

Добавить CSS-блок для адаптации вёрстки к печати:

```css
@media print {
  body {
    background: white;
    padding: 0;
  }

  .container {
    box-shadow: none;
    max-width: 100%;
  }

  header {
    background: white;
    color: #343a40;
    padding: 20px;
    border-bottom: 3px solid #667eea;
  }

  /* Скрыть элементы управления */
  .back-btn,
  .results-tabs,
  .export-buttons,
  .scoring-total-card {
    display: none !important;
  }

  /* Показать только активную вкладку */
  .tab-content {
    display: block !important;
    padding: 20px;
  }

  .tab-content:not(.active) {
    display: none !important;
  }

  /* Запретить разрыв элементов */
  .category-block,
  .requirement-item,
  .rec-category-block,
  .recommendation-item {
    break-inside: avoid;
  }

  /* Убрать тени у карточек */
  .summary-card,
  .principle-score-card {
    box-shadow: none;
    border: 1px solid #ddd;
  }

  /* Сохранить цвета при печати */
  .requirement-item.compliant,
  .summary-card.compliant {
    background: #d4edda !important;
    -webkit-print-color-adjust: exact;
    print-color-adjust: exact;
  }

  .requirement-item.violation,
  .summary-card.violations {
    background: #f8d7da !important;
    -webkit-print-color-adjust: exact;
    print-color-adjust: exact;
  }

  /* Заголовок для печати */
  .print-header {
    display: block !important;
    text-align: center;
    margin-bottom: 30px;
    padding: 20px;
    border-bottom: 2px solid #343a40;
  }

  .print-header h1 {
    margin: 0 0 10px 0;
    color: #343a40;
  }

  .print-header p {
    margin: 5px 0;
    color: #6c757d;
  }
}

.print-header {
  display: none;
}
```

### 3. Печать конкретного элемента (сертификат, документ)

Функция для печати отдельного элемента в новом окне:

```javascript
window.printCertificate = function() {
  const content = document.getElementById('certificateContent').innerHTML;
  const printWindow = window.open('', '_blank');
  printWindow.document.write(`
    <!DOCTYPE html>
    <html>
    <head>
      <title>Сертификат соответствия</title>
      <style>
        body { font-family: Arial, sans-serif; padding: 40px; }
        @media print { body { padding: 0; } }
      </style>
    </head>
    <body>${content}</body>
    </html>
  `);
  printWindow.document.close();
  setTimeout(() => printWindow.print(), 250);
};
```

Кнопка для вызова:

```html
<button onclick="printCertificate()" style="padding: 10px 20px; background: #667eea; color: white; border: none; border-radius: 8px; cursor: pointer;">
  🖨️ Распечатать сертификат
</button>
```

### 4. Опционально: динамический заголовок для печати

Функция для заполнения данных перед печатью:

```javascript
function updatePrintHeader() {
  const printCharterName = document.getElementById('printCharterName');
  const printDate = document.getElementById('printDate');

  if (printCharterName) {
    printCharterName.textContent = charterName || 'Устав потребительского кооператива';
  }
  if (printDate) {
    printDate.textContent = 'Дата анализа: ' + new Date().toLocaleDateString('ru-RU');
  }
}
```

HTML-блок заголовка:

```html
<div class="print-header">
  <h1>Анализ устава кооператива</h1>
  <p id="printCharterName"></p>
  <p id="printDate"></p>
</div>
```

1. Скопировать CSS-блок `@media print` в основной файл стилей
2. Добавить кнопку печати с `onclick="window.print()"`

4. При необходимости — добавить `.print-header` с динамическими данными
