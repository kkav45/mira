## 🔄 ПОРЯДОК ДЕЙСТВИЙ ДЛЯ ПОЛЬЗОВАТЕЛЯ

1. **Очистить кэш браузера:**
   - Windows: `Ctrl + Shift + Delete`
   - Mac: `Cmd + Shift + Delete`
   - Или: Hard Reload (`Ctrl + F5` / `Cmd + Shift + R`)

2. **Проверить шаги:**
   - Шаг 1: Маршрут → Анализ → ✅ Кнопка "Далее" активна
   - Клик "Далее" → Шаг 2: Пилот
   - Клик "Далее" → Шаг 3: Отчёт
   - **Шага 4 нет!**

3. **Проверить дашборд:**
   - После анализа на шаге 1 кнопка "ДАШБОРД" становится синей
   - Клик → Открывается модальное окно на весь экран
